Описание директорий текущего каталога.

Bin\      -  бинарный файл штатной библиотеки Lusbapi.dll

Include\  -  файлы прототипов для С++

Lib\      -  библиотеки импорта и модули объявлений для различных сред программирования

Source\   -  директория с исходными текстами.
 |
 |_ Lusbapi\ - исходные тексты штатной библиотеки Lusbapi.dll. 
               Проект собран в среде Borland С++ 5.02


CopyLusbapi.bat   - командный файл копирования библиотеки Lusbapi.dll в системную 
                    директорию

RemoveLusbapi.bat - командный файл удаления библиотеки Lusbapi.dll из системной 
                    директории
